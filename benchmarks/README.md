# Benchmarks

Monorepo benchmark baselines generated by Vitest bench.

## Run

```bash
pnpm bench -- --label=2026-04-17-baseline
```

You can replace the label with any stable name, for example:

- `2026-04-17-baseline`
- `before-optimization`
- `after-regex-refactor`

## Output

- `benchmarks/<label>/SUMMARY.md`: cross-package summary
- `benchmarks/<label>/manifest.json`: machine-readable run manifest
- `packages/*/benchmarks/<label>.md`: per-package markdown report
- `packages/*/benchmarks/<label>.json`: raw Vitest benchmark output
